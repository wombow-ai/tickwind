## Top 6(带月成本)

1. **个股 AI 速览(每日缓存摘要)** — 5 条带来源角标的中文要点(新闻+社媒+公告+信号)。50 票×30 天 ≈ **$0.45/月**(源哈希跳过后);批量层换 GLM → $0。长尾票首次浏览触发+缓存 24h。`/v1/stocks/{t}/summary` 端点已存在。努力 M。【Robinhood Cortex Digests / moomoo AI Summary 同形态——**卡片化缓存摘要,不是聊天**】
2. **"为什么涨/跌?"异动解释器** — 仅 |涨跌|≥5% 且被浏览时触发,2-3 句+证据链;无催化剂输出固定话术。≈**$0.10/月**。幻觉风险最高→硬护栏:引用校验失败回退 canned 话术、禁词表(目标价/买卖)。努力 S–M(搭 #1 管线)。
3. **AI 中文每日盘前简报(+周报)** — 一次生成=全体用户+SEO。≈**$0.03/月**。输入全是自家结构化数据,数字回查即可。挂 /briefing+首页模块。努力 S–M。**Token 经济性之王**。
4. **英文新闻标题中译** — ingest 时批量 50 条/请求,存 news 表新列,中文界面显示+悬停原文+「AI 译」角标。LLM ≈ $0.14/月(GLM → $0);对比:DeepL 免费 50 万字符/月不够,Azure F0 够但多 vendor 且金融术语弱→**用 LLM**(可挂术语表当系统词缓存)。标题按 news id **永久缓存**。努力 S。
5. **自然语言选股** — LLM 只做解析器:中文 query → JSON 筛选参数 → 现有 screenQuotes;解析结果以可编辑 chips 回显。≈**$0.03/月**(1000 次查询)。无散文输出=无幻觉叙事面。依赖市值缓存(#5b)。差异化:能筛 DTC/内部人买入/WSB 热度等独有信号。努力 M。
6. **财报/公告 AI 摘要(8-K 业绩稿 + 10-K 章节)** — filing 不可变→键 (accession, section, prompt_ver) **永久缓存**。稳态 ≈$0.12/月;回填 50 票最新 10-K 章节一次性 $0.35。**财报电话会转录稿无合规免费源(RED)→用 8-K 新闻稿替代**。提取式摘要带原文短引。SEC 公域,合规最干净。努力 M–L(章节解析是真功夫,放最后)。

**第 7/8 处理**:组合 AI 点评→**不调 LLM**,用 #1 缓存按持仓确定性拼装(避开个性化建议合规雷)。评论 TL;DR→暂缓(评论量近零)。
